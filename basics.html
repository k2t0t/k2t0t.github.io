basic commands

the fundamentals

if you got here, you probably already know how to use the ls and cd commands:

  • ls - lists directory contents -> ls directory_path
    • directories are basically folders that contain other files and directories and help organize the filesystem
    • you can add a flag like -la to the ls command (ls -la) to access more information like hidden files (with a . in front of them) and file permissions
  • cd - changes directory -> cd directory_path
    • you can use cd .. to go up one directory (go backwards)
    • pwd shows your current directory
    • it's important to utilize the correct filepath when using cd, either by using the full path (eg. cd /home/user/documents) or a path relative to your current directory (eg. you are in /home/user and you cd to documents).

    these commands are essential for navigating the filesystem!

    just as important are the commands for accessing and modifying information -- here are some basics:

  • cat - displays file contents -> cat file.txt
  • nano - basic text editor for a file -> nano file.txt (you can also use vim)
  • touch - creates an empty file -> touch file.txt
  • mkdir - creates a directory -> mkdir directory_name
  • rm - removes a file or directory -> rm file.txt or rm -r directory_name (note: removes directory contents)
  • cp - copies a file or directory -> cp source_path destination_path
  • mv - moves a file or directory -> mv source_path destination_path

    • it looks like a lot but it's good to eventually be familiar with these commands!
